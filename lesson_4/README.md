# Задачи

Здесь работаем с теми же четырьмя txt файлами, которые были на прошлом занятии

## Задача №1

**Сервер:**
1) С каждым клиентом работает в отдельном потоке, чтобы обрабатывать несколько клиентов одновременно
2) Клиент отправляет команду:
```
/get <filename>
```
3) Сервер проверяет наличие файла:
- Если файл есть → отправляет его содержимое клиенту
- Если файла нет → отправляет: "Файл <filename> не найден"
4) После отправки сервер ждёт следующей команды клиента (или /exit)

**Клиент:**
1) Отправляет команду
```
/get <filename>
```
2) Получает содержимое файла и выводит его в консоль
3) Может отправлять несколько команд подряд
4) Команда /exit закрывает соединение

## Задача №2

1) С каждым клиентом работает в отдельном потоке, чтобы обрабатывать несколько клиентов одновременно
2) Клиент отправляет команду:
```
/count <filename>
/lines <filename>
/words <filename>
/exit
```
3) Сервер открывает нужный файл и:
- /count → возвращает количество символов
- /lines → возвращает количество строк
- /words → возвращает количество слов
