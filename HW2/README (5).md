## Condition

**Condition -** —ç—Ç–æ –±–æ–ª–µ–µ —Å–ª–æ–∂–Ω—ã–π –º–µ—Ö–∞–Ω–∏–∑–º, –∫–æ—Ç–æ—Ä—ã–π –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç Lock + Event + –ª–æ–≥–∏—á–µ—Å–∫–æ–µ —É—Å–ª–æ–≤–∏–µ. –ü–æ—Ç–æ–∫–∏ –∂–¥—É—Ç –Ω–µ –ø—Ä–æ—Å—Ç–æ —Å–∏–≥–Ω–∞–ª–∞, –∞ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–≥–æ —É—Å–ª–æ–≤–∏—è

**–ü—Ä–∏–Ω—Ü–∏–ø —Ä–∞–±–æ—Ç—ã:**
1. –ü–æ—Ç–æ–∫ –∑–∞—Ö–≤–∞—Ç—ã–≤–∞–µ—Ç Condition (–∫–∞–∫ Lock)
2. –ü—Ä–æ–≤–µ—Ä—è–µ—Ç —É—Å–ª–æ–≤–∏–µ - –µ—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ –Ω–µ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –≤—ã–∑—ã–≤–∞–µ—Ç wait()
   * wait() –≤—Ä–µ–º–µ–Ω–Ω–æ –æ—Ç–ø—É—Å–∫–∞–µ—Ç Lock –∏ —É—Å—ã–ø–ª—è–µ—Ç –ø–æ—Ç–æ–∫ 
3. –î—Ä—É–≥–æ–π –ø–æ—Ç–æ–∫, –∏–∑–º–µ–Ω–∏–≤ —Å–æ—Å—Ç–æ—è–Ω–∏–µ, –≤—ã–∑—ã–≤–∞–µ—Ç notify() –∏–ª–∏ notify_all()
   * notify() - –±—É–¥–∏—Ç –æ–¥–∏–Ω —Å–ª—É—á–∞–π–Ω—ã–π –ø–æ—Ç–æ–∫ –∏–∑ –æ—á–µ—Ä–µ–¥–∏ –æ–∂–∏–¥–∞–Ω–∏—è
   * notify_all() - –±—É–¥–∏—Ç –≤—Å–µ –ø–æ—Ç–æ–∫–∏ –∏–∑ –æ—á–µ—Ä–µ–¥–∏ –æ–∂–∏–¥–∞–Ω–∏—è
4. –ü—Ä–æ—Å–Ω—É–≤—à–∏–µ—Å—è –ø–æ—Ç–æ–∫–∏ —Å–Ω–æ–≤–∞ –ø—ã—Ç–∞—é—Ç—Å—è –∑–∞—Ö–≤–∞—Ç–∏—Ç—å Lock –∏ –ø—Ä–æ–≤–µ—Ä—è—é—Ç —É—Å–ª–æ–≤–∏–µ


**–ú–µ—Ç–æ–¥—ã Condition**
```python

    while not condition_met:
        condition.wait()           # –ñ–¥–∞—Ç—å –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ
        condition.wait(timeout=5)  # –ñ–¥–∞—Ç—å –º–∞–∫—Å–∏–º—É–º 5 —Å–µ–∫—É–Ω–¥

    condition.notify()    # –†–∞–∑–±—É–¥–∏—Ç—å 1 —Å–ª—É—á–∞–π–Ω—ã–π –ø–æ—Ç–æ–∫
    condition.notify(2)   # –†–∞–∑–±—É–¥–∏—Ç—å 2 —Å–ª—É—á–∞–π–Ω—ã—Ö –ø–æ—Ç–æ–∫–∞

with condition:
    condition.notify_all()  # –†–∞–∑–±—É–¥–∏—Ç—å –≤—Å–µ –∂–¥—É—â–∏–µ –ø–æ—Ç–æ–∫–∏
```

# Queue

–ú—ã –≥–æ–≤–æ—Ä–∏–ª–∏ —Å –≤–∞–º–∏ –æ RaceCondition (—Å–æ—Å—Ç–æ—è–Ω–∏–µ –≥–æ–Ω–∫–∏) –∏ –ø–æ–º–Ω–∏–º, —á—Ç–æ –º—É—á–∞—Ç—å—Å—è —Å –Ω–∏–º –Ω—É–∂–Ω–æ –º–æ—â–Ω–æ, —Ç–∞–∫ –∫–∞–∫ –¥–ª—è –ª—é–±–æ–π –º–∏–∫—Ä–æ—Ä–∞–±–æ—Ç—ã —Å –æ–±—â–∏–º —Ä–µ—Å—É—Ä—Å–æ–º –Ω–∞–º –Ω–∞–¥–æ –ø–æ—Å—Ç–æ—è–Ω–Ω–æ —Å—Ç–∞–≤–∏—Ç—å –±–ª–æ–∫–∏—Ä–æ–≤–∫—É

**–û—Å–Ω–æ–≤–Ω—ã–µ –º–µ—Ç–æ–¥—ã:**
